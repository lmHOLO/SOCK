```
🧾 요약

우리꺼 메인 repo : upstream

각자 fork한 repo : origin

### **UPSTREAM 관련 GIT**

1. upstream에 있는 내용 합칠 브랜치에 checkout : git checkout 브랜치명
2. upstream에서 원하는 브랜치 fetch + merge 하는 명령어 : git pull upsteam 브랜치명
3. 내 레포지토리에 올릴 명령어 : git push origin 현재브랜치명
```
